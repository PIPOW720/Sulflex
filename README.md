# SULFLEX — Manutenção Industrial e Predial

Site institucional + Catálogo de 61 Equipamentos de Climatização.

## Como usar
1. Extraia o ZIP
2. Abra `index.html` no navegador

## Páginas
- `index.html` — Home (slider, stats, serviços, mapa)
- `portfolio.html` — ⭐ Catálogo Loja (61 produtos, 7 categorias, busca/filtro)
- `quem-somos.html` — Quem Somos (timeline 2015-2025, MVV)
- `contato.html` — Contato (formulário de orçamento)
- `trabalhe-conosco.html` — Carreiras
- `servicos.html` — Serviços (HVAC, Elétrica, Hidráulica, Pintura, Reformas, Preventiva)
- `presenca.html` — Localização (Paranaguá/PR + mapa)
